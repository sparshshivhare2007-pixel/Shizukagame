## Deploying your own

### To Heroku

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/TheChampu/CutieGame)

### On your machine

1. Install requirements:

```bash
pip install -Ur requirement.txt
```

2. Copy `.env.example` to `.env` and add your credentials.
3. Run `main.py`:

```bash
python main.py
```
